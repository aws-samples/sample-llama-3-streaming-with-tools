Amazon Bedrock Llama 3 Examples

This project demonstrates three different approaches to working with Amazon Bedrock's Llama 3 model. Select one of the examples below to explore different capabilities.

1. Llama using Tools Non-streaming implementation using the Converse API for tool use 2. Llama using Streaming Real-time streaming responses using Server-Sent Events 3. Llama using Streaming and Tools Combined approach with streaming responses and tool use
